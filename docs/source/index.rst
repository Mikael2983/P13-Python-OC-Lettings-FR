OC_lettings documentation
=========================

Welcome to the **OC_lettings** documentation.
This project is a Django application developed as part of the **OpenClassrooms Python Developer** path.

The application manages:
- User profiles
- Rental listings
- Main site configuration

It is designed to be easily deployed on **Render**, via a Docker container available on **DockerHub**,
and benefits from continuous integration using **GitHub Actions**.
